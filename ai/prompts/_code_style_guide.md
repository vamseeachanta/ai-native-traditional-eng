What is the best package or guidelines to reference here
python
- PEP 8: The Style Guide for Python Code
- PEP 257: Docstring Conventions
- Black: The Uncompromising Code Formatter
- isort: A Python utility for sorting imports
- flake8: A tool for enforcing coding style in Python
- mypy: An optional static type checker for Python

Markdown
- Common Markdown style guides (e.g., CommonMark, GitHub Flavored Markdown)
- Markdown linters (e.g., markdownlint)
- Prettier: A code formatter that supports Markdown and many other languages
- remark: A Markdown processor powered by plugins
- markdown-it: A Markdown parser that can be extended with plugins
- markdownlint: A Markdown linter for enforcing style guidelines
- markdown-toc: A tool for generating table of contents in Markdown files
